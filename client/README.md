# SA-Services Client

This is a simple client to consume the SA-Services. It is the frontend application that consumes all these microservices.

Runs on `localhost:8000` and `172.10.10.5:80`.


### Endpoints

| Route            |
| ---------------- |
| `/login`         |
| `/logout`        |
| `/register`      |
| `/events`        |
| `/subscriptions` |
| `/checkin`       |
| `/checkin/:id`   |
